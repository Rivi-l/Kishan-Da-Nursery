⚙ One-time setup needed — host this file so QR codes work
  1. Go to github.com → New repository (name it anything, e.g. kishan-nursery) → set to Public
  2. Upload tree.html to that repository
  3. Go to repo Settings → Pages → Source: main branch → Save
  4. Your URL will be: https://YOUR-USERNAME.github.io/kishan-nursery/tree.html?id=1
  5. In the Admin QR tool, set the Base URL to that address — then regenerate all QR codes

After hosting, anyone who scans a QR code will land directly on this page with the correct tree info. No server costs, no subscription — GitHub Pages is free forever.
